MobileProof Mobile Application

The most recent project I've worked on is an application called MobileProof which is supposed to implement a functional mobile form of the website proofwiki. I was working with my friend Madhav Kumar (@MadhavRKumar on Github), and I was solely responsible for all of the server side aspects of creating the app. This is because we didn't have the money to buy a dedicated server, so we turned an old Dell Inspiron into an Ubuntu 16.04 server. From there, I used python network programming to make a threaded server that would be able to handle search requests.
